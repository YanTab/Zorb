# Documentation du projet Zorb Earth Run

Ce dossier contient les documents de conception et de planification.

Fichiers principaux :
- `GCD.md` - Game Concept Document
- `GFD.md` - Game Features Document
- `TDD.md` - Technical Design Document
- `Roadmap.md` - Roadmap détaillée
- `Conventions.md` - conventions de nommage / workflow.

## Processus de mise à jour
- Mettre à jour chaque document avec la version + date en en-tête.
- Conserver l'historique dans Git (branch, commit par version).
